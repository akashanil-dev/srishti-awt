# Project Overview

## Introduction
The **Online Event Registration Portal** is a responsive web-based application designed to streamline and modernize the process of managing event registrations for a college technology festival ("Srishti"). This digital solution replaces manual registrations, reducing errors and long queues while providing an interactive platform for students.

## Key Features

1. **Event Listing & Discovery**
   - **Bootstrap Card Layout**: Displays all available events in an organized, visually appealing grid.
   - **Event Details**: Each card includes the title, a short description, and the number of available seats.

2. **Seamless Registration Process**
   - **Modal Form Registration**: Students can click the "Register" button to open a modal form without leaving the current page.
   - **Form Validation**: Uses jQuery for client-side validation to ensure email and phone number correctness before submission.
   - **AJAX Submission**: Submits data securely to the server asynchronously without reloading the page, giving immediate feedback.

3. **Real-time Updates & Constraints**
   - **Seat Tracking**: Dynamically tracks and updates available seats as users register.
   - **Automatic Disabling**: Prevents overbooking by disabling the registration button when an event is full.

4. **Participant Management**
   - **Participant List**: Dynamically loads and displays registered participants using AJAX in real time.
   - **Team Registration**: Supports registration for both solo and team events, tracking team members.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, Bootstrap, jQuery, AJAX
- **Backend**: PHP 
- **Database**: MySQL
